隱私權政策

最後更新日期：2026 年 6 月 23 日

TripApp（以下簡稱「本應用程式」）由開發者 David Yu（以下簡稱「我們」）開發及營運。我們重視您的隱私，本政策說明我們如何收集、使用、儲存及保護您的個人資料。

一、我們收集哪些資料

1. 帳號資料

  • 電子郵件地址（用於註冊和登入）

2. 使用者產生的內容

  • 行程規劃資料（行程名稱、目的地、日期、景點、交通、住宿、預算等）
  • 收藏的景點資料（名稱、地址、分類、備註）
  • 上傳的照片（景點照片、機票憑證、住宿憑證等）

3. 位置資料

  • 裝置的 GPS 位置（僅在您授權後使用，用於在地圖上顯示您的所在位置及附近景點）

4. 自動收集的技術資料

  • 裝置類型、作業系統版本（用於改善應用程式相容性）

二、我們如何使用您的資料

  • 提供行程規劃、景點收藏、地圖顯示等核心功能
  • 透過 AI 服務（Claude API）為您生成行程建議、辨識景點
  • 透過 Google Maps / Google Places API 提供地圖及景點搜尋功能
  • 改善應用程式的功能與使用體驗

三、第三方服務

本應用程式使用以下第三方服務，您的部分資料可能會傳送至這些服務：

  • Supabase — 資料儲存與帳號驗證（資料儲存於雲端伺服器）
  • Google Maps / Google Places API — 地圖顯示與景點搜尋（傳送搜尋關鍵字與位置資訊）
  • Anthropic Claude API — AI 行程生成與景點辨識（傳送行程相關文字資料）

這些第三方服務各有其隱私權政策，建議您參閱：

  • Supabase 隱私權政策
  • Google 隱私權政策
  • Anthropic 隱私權政策

四、資料儲存與安全

  • 您的資料儲存於 Supabase 提供的雲端資料庫，採用行級安全策略（Row Level Security），確保只有您本人能存取自己的資料
  • 上傳的照片儲存於 Supabase Storage，採用簽章 URL 存取控制
  • 我們不會將您的個人資料出售或出租給任何第三方

五、您的權利

您隨時可以：

  • 在應用程式內查看、編輯或刪除您的行程與收藏資料
  • 撤回位置存取權限（透過裝置的「設定」→「隱私權與安全性」→「定位服務」）
  • 要求我們刪除您的帳號及所有相關資料（請透過下方聯絡方式聯繫我們）

六、兒童隱私

本應用程式不針對 13 歲以下兒童設計，我們不會故意收集 13 歲以下兒童的個人資料。若您發現有兒童的資料被收集，請聯繫我們，我們會立即刪除。

七、隱私權政策的變更

我們可能會不定期更新本隱私權政策。變更後的政策將發佈於本頁面，並更新「最後更新日期」。建議您定期查看本政策。
